# Move, Play & Belong project website

Static website template for the Erasmus+ Sport project **Move, Play & Belong: Inclusive Family Sports**.

## Sections included

- About the project
- News
- Guidance Materials
- Partners
- Contacts
- EU co-funded logo and disclaimer

The top navigation intentionally includes only:

- About the project
- News
- Guidance Materials

## How to use

Open `index.html` locally in a browser, or deploy the whole folder as a static website on Vercel, Netlify, GitHub Pages or any web hosting service.

## Content to update

- Replace `add-project-email@example.eu` in `index.html` with the real project contact email.
- Replace example news cards with actual project updates.
- Link the Guidance Materials cards to final PDFs, online tools or videos once available.
- If strict EU visual identity compliance is required, replace `assets/img/eu-cofunded.svg` with the official co-funded EU logo file.
